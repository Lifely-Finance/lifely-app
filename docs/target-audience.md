# Портрет целевого пользователя Lifely

### 1. Профиль "Амбициозный профессионал"
* **Возраст:** 22–35 лет.
* **Доход:** $1,500 – $5,000+.
* **Проблема:** Доход растет, но капитал не копится («деньги сквозь пальцы»).
* **Цель:** Сформировать первый значимый капитал ($50k–$100k), выйти из операционки трат.

### 2. Ключевые "Боли", которые решает Lifely
* **Хаос в данных:** Информация о жизни и деньгах разбросана по 5 приложениям.
* **Отсутствие фокуса:** Пользователь не понимает, как чашка кофе сегодня влияет на его квартиру через 5 лет.
* **Слабая дисциплина:** Сложно поддерживать систему учета дольше двух недель.

### 3. Почему он выберет Lifely?
Lifely дает не просто сухие цифры, а **"Цифрового двойника"**, который подсказывает: "Если ты сохранишь этот темп, твоя цель будет достигнута на 4 месяца раньше".
