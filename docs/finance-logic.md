# Финансовая логика Lifely (Wealth Architect)

Система базируется на трех математических столпах, которые обеспечивают экспоненциальный рост капитала пользователя.

### 1. Расчет «Чистой Дельты»
Формула: `Delta = Income - Expenses`. 
В Lifely дельта — это не просто остаток, а «топливо» для инвестиций. Система анализирует волатильность трат и предлагает оптимальный процент удержания.

### 2. Модель сложных процентов (Compound Interest)
Приложение визуализирует путь к цели через формулу:
$$A = P(1 + r/n)^{nt}$$
Где пользователь видит реальное влияние ежедневной дисциплины на капитал через 5-10 лет.

### 3. Анализ «Скрытых потерь»
Алгоритм выявляет микро-траты, которые незаметно уничтожают потенциальный капитал, и пересчитывает их в «стоимость упущенной выгоды» через 20 лет.
